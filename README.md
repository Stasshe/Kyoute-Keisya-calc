# Kyoute Keisya Calc

共通テストの点数を入力して、大学ごとの傾斜配点（重み）を計算するシンプルな静的サイトです。

概要
- **目的**: 共通テストの点数から、大学・学部・学科ごとの傾斜配点を計算して合計点を出すためのツール。
- **デザイン**: 企業的でわかりやすくシンプルなUI。ライトテーマで固定、色はハードコーディングされています。
- **静的サイト**: Next.js を使った静的構成（動的パラメータは使用しない）。Tailwind CSS v3 を使用。

主な特徴
- **理系／文系対応**: 教科構成に合わせて計算が可能。
- **ローカル保存**: 入力値やカスタムの設定はブラウザの `localStorage` に保存されます。
- **カスタム配点**: 各大学のデフォルト配点を使えるほか、ユーザーがカスタムを追加・編集できます。
- **レスポンシブ**: モバイル〜デスクトップに対応。

教科（実装上の扱い）
- 社会①、社会②
- 国語（/200）
- 英語R（リーディング）
- 英語L（リスニング）
- 理科①、理科②
- 数学①、数学②
- 情報

配点（weight）について
- `weight` は実際の配点数値を入れます（例: 阪大の英語R 37.5、英語L 12.5 のように実点を指定）。

入力ルール・注意点
- 入力欄は空白を許容します。空白は `0` と見なします。
- 共通テストの満点は固定（内部で扱うためユーザーが入力する必要はありません）。
- `themecontext` は削除して問題ありません（ライトテーマで統一しています）。

データとファイル構成（主要箇所）
- 傾斜配点データ: `src/data/universities.ts`
- 主なコンポーネント: `src/components/Calculator.tsx`, `UniversityList.tsx`, `UniversityEditor.tsx`, `DepartmentEditor.tsx`

開発者向け: ローカル実行
1. 依存関係をインストール
```
npm install
```
2. 開発サーバーを起動
```
npm run dev
```
3. ビルド
```
npm run build
```

その他
- Tailwind v3 を使用しています。
- サイトは静的出力を前提にしています（動的ルーティングは使用していません）。

貢献 / 変更のガイドライン
- 大きなデザイン変更は避け、企業的で落ち着いた配色を保ってください。
- 不要なアニメーションは控えめに。
